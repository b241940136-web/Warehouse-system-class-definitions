# Агуулахын Систем — Warehouse Management System

## Шаардлагын товч тайлбар

Байгууллага олон агуулахтай. Агуулах бүр няравтай бөгөөд нярав:
- Барааг орлогот хүлээн авч, орлогын падаан үйлдэнэ
- Барааг зарлагаар гаргаж, зарлагын падаан үйлдэнэ
- Нөөцийн тайлан болон няравын тайланг харна
- Тооллого хийж илүүдэл/дутагдлыг бүртгэнэ

---

## Нэр үг / Үйл үгийн шинжилгээ

### Нэр үгүүд (Классууд)
| Нэр үг | Класс |
|--------|-------|
| Агуулах | `Warehouse` |
| Нярав | `Storekeeper` |
| Бараа | `Product` |
| Нөөц | `Stock` |
| Орлогын падаан | `ReceiptVoucher` |
| Зарлагын падаан | `IssueVoucher` |
| Падааны мөр | `VoucherItem` |
| Нөөцийн тайлан | `StockReport` |
| Няравын тайлан | `StorekeeperReport` |
| Тооллогын бичлэг | `InventoryCount` |

### Үйл үгүүд (Методууд)
| Үйл үг | Метод | Класс |
|--------|-------|-------|
| Томилох | `assignStorekeeper()` | Warehouse |
| Хүлээн авах | `receiveGoods()` | Storekeeper |
| Гаргах | `issueGoods()` | Storekeeper |
| Нөөцийн тайлан харах | `viewStockReport()` | Storekeeper |
| Няравын тайлан харах | `viewStorekeeperReport()` | Storekeeper |
| Тооллого хийх | `performInventoryCount()` | Storekeeper |
| Нэмэх | `increaseQuantity()` | Stock |
| Хасах | `decreaseQuantity()` | Stock |
| Хэвлэх | `print()` | ReceiptVoucher, IssueVoucher, InventoryCount |
| Харуулах | `display()` | StockReport, StorekeeperReport |
